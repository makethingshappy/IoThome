# Tasmota Templates — IoTsmart ESP32-S3

Tasmota templates for IoTextra mezzanine modules used with the **IoTsmart ESP32-S3** platform.

## Available Templates

|------------------|-----------------|

| SSR Small        | ✅ Available    |
| MOSFET2          | ✅ Available    |
| Octal            | ✅ Available    |

| Analog           | 🔶 Coming Soon  |
