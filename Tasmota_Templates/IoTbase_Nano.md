# Tasmota Templates — IoTbase Nano

Tasmota templates for IoTextra mezzanine modules used with the **IoTbase Nano** platform.

## Available Templates

|------------------|-----------------|
| Input            | ✅ Available    |
| Relay2           | ✅ Available    |
| SSR Small        | ✅ Available    |
| MOSFET2          | 🔶 Coming Soon  |

| Analog           | ✅ Available    |
| Analog2          | 🔲 Planned      |
| Analog3          | 🔶 Coming Soon  |
| Combo            | ✅ Available    |
